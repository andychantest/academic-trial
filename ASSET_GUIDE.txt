說明 / Notes：
  所有資產均為「可選」。/ All assets are "optional".
  若對應檔案不存在，遊戲會自動使用內建幾何圖形/預設顏色代替。
  If a file doesn't exist, the game uses built-in graphics/colors automatically.
  只需將圖片命名為正確名稱並放入正確資料夾，重新整理瀏覽器即生效。
  Just name files correctly and place in the right folder, then refresh the browser.
  無需修改任何程式碼。/ No code changes needed.

重要更新 / Important Update：
  遊戲使用 PNG 序列（多張圖片）實現動畫效果，而非 GIF。
  Game uses PNG sequence (multiple images) for animation, not GIF.
  詳情請參閱「第九節：動畫系統」。
  See "Section IX: Animation System" for details.

📁 九、動畫系統 / ANIMATION SYSTEM
   資料夾：所有圖片資料夾
═══════════════════════════════════════════════════════════════════

動畫原理 / How Animation Works：
  使用多張 PNG 圖片代替 GIF，程式自動偵測並循環播放。
  系統根據 config.txt 中的 animation_fps 參數控制播放速度。
  Use multiple PNG images instead of GIF for smooth animations.
  System reads animation_fps from config.txt to control playback speed.

命名規範 / Naming Convention：
  {type}_{name}_{number}.png
  數字從 1 開始，程式自動偵測最大數字。
  Numbers start from 1, program auto-detects max frame number.

玩家角色動畫 / Player Animation：
  格式 / Format: player_{role}_{state}_{number}.png
  範例 / Examples:
    player_student_idle_1.png     學生待機第1幀 / Student idle frame 1
    player_student_idle_2.png     學生待機第2幀 / Student idle frame 2
    player_student_idle_3.png     學生待機第3幀 / Student idle frame 3
    player_student_move_1.png     學生移動第1幀 / Student move frame 1
    player_student_move_2.png     學生移動第2幀 / Student move frame 2
    player_student_move_3.png     學生移動第3幀 / Student move frame 3

敵人動畫 / Enemy Animation：
  格式 / Format: enemy_{id}_{state}_{number}.png
  範例 / Examples:
enemy_plagiarism_1.png        抄襲第1幀 / Plagiarism frame 1

    enemy_plagiarism_2.png        抄襲第2幀 / Plagiarism frame 2

    enemy_plagiarism_fast_1.png   抄襲加速第1幀 / Plagiarism fast frame 1

    enemy_plagiarism_fast_2.png   抄襲加速第2幀 / Plagiarism fast frame 2

收集物動畫 / Collectible Animation：
  格式 / Format: collectible_{id}_{number}.png
  範例 / Examples:
    collectible_journal_1.png   期刊第1幀 / Journal frame 1
    collectible_journal_2.png   期刊第2幀 / Journal frame 2

說明 / Notes：
  - 只有 1 張圖片時顯示靜態 / Single image = static (no animation)
  - 2 張或以上自動播放動畫 / 2+ images = automatic animation
  - 數字必須連續 / Numbers must be consecutive (1, 2, 3...)
  - 建議每個動畫 2-4 幀 / Recommended 2-4 frames per animation

config.txt 設定 / config.txt Settings：
  [ANIMATION]
  animation_fps = 10    # 每秒幀數 (建議值 8~15)

⚠️ 重要提示 / IMPORTANT NOTES
═══════════════════════════════════════════════════════════════════

1. 檔名大小寫敏感 / Filenames are case-sensitive
   正確 / Correct：enemy_plagiarism_1.png
   錯誤 / Wrong：Enemy_Copycat_1.PNG

2. 動畫優先順序 / Animation Priority：
   多張 PNG (player_*_1.png, *_2.png...) → 自動播放動畫
   單張 PNG → 顯示靜態
   都沒有 → 用幾何圖形

3. 音效優先順序 / SFX Priority：
   .mp3 存在 → 用 .mp3 / .mp3 → use .mp3
   不存在 → 靜音 / None → silent

4. 圖片格式建議 / Image Format Recommendations：
   透明背景 → PNG / Transparent → PNG
   全畫面背景 → JPG / Full screen → JPG
   動畫 → PNG 序列 (建議) / Animation → PNG sequence (recommended)

5. 尺寸建議 / Size Recommendations：
   過場和背景：960×540（16:9）/ Cutscenes/BG: 960×540 (16:9)
   玩家角色：64×64 / Player: 64×64
   敵人：48×48 / Enemy: 48×48
   收集物：32×32 / Collectible: 32×32

6. 更新資產後只需重新整理瀏覽器（F5），不需要修改任何程式碼。
   After adding assets, just refresh (F5). No code changes needed.
